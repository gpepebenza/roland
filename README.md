# Roland Landing Page

This repository contains a static single-page site for GripNex. Open `index.html` in your browser or start a simple server and visit the link below.

## Local link
1. Start a local server from the project root:
   ```bash
   python -m http.server 8000
   ```
2. Open the site at: http://localhost:8000/

The navigation includes a **Home** link that returns to the top of the page.
